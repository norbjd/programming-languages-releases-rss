Programming languages releases RSS feeds

The RSS feeds under that GitHub Pages site are generated from https://github.com/norbjd/programming-languages-releases-rss.

There is one RSS feed per programming language.

To know about available languages, take a look at the gh-pages branch.

For example, for Rust, just go to https://norbjd.github.io/programming-languages-releases-rss/rust.xml.

You can then subscribe to your favorite languages feeds using any RSS reader.
